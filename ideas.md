# Adaptive STEM Learning Platform - 设计头脑风暴

## 项目概述
将一个基于 Python Streamlit 的自适应 STEM 学习应用转化为现代化 Web 应用。核心功能包括：自适应选题引擎（Active Inference + 4PL IRT）、间隔复习、知识图谱、成就系统、限时挑战、学习报告等。

---

## Idea 1: "Neural Topology" — 神经拓扑主义

**Design Movement**: 受数据可视化艺术和神经科学图谱启发的有机技术美学

**Core Principles**:
1. 有机网络结构 — 所有布局遵循神经网络拓扑，节点与连接线构成视觉骨架
2. 生物发光色彩 — 深色背景上的荧光色彩，模拟神经元放电的视觉效果
3. 流动状态 — 界面元素随学习状态动态变化，如同活体神经网络

**Color Philosophy**: 深邃的墨蓝底色（#0A0E1A）搭配电离蓝（#00D4FF）和突触绿（#00FF94）的荧光高亮。错误用珊瑚红（#FF4757），成就用琥珀金（#FFD700）。色彩传达"大脑在线"的感觉。

**Layout Paradigm**: 中心辐射式布局。主学习区域居中，各功能模块如神经元般围绕分布。侧边栏是一条"神经脊柱"，导航项如突触节点。

**Signature Elements**:
1. 粒子连接线动画 — 页面背景有微弱的神经网络粒子效果
2. 脉冲进度指示器 — XP 和能力值用脉冲波形展示
3. 突触过渡动画 — 页面切换时元素沿神经路径流动

**Interaction Philosophy**: 每次正确回答触发"突触放电"微动画，错误回答触发"信号衰减"效果。悬停时节点发光扩散。

**Animation**: 缓慢的呼吸式背景脉动（4s周期），元素入场用弹性缓动（spring），数据图表用描边动画逐步绘制。

**Typography System**: 标题用 Space Grotesk（几何感强），正文用 IBM Plex Sans（技术感但可读性好），数据用 JetBrains Mono（等宽，精确感）。

## Idea 2: "Ink & Paper" — 学术手稿风

**Design Movement**: 受 Dieter Rams 极简主义和学术论文排版启发的知识密集型设计

**Core Principles**:
1. 信息密度优先 — 像学术论文一样高效利用空间，减少装饰性元素
2. 排版即设计 — 通过字体大小、粗细、间距创造视觉层次，而非颜色和图形
3. 留白即呼吸 — 精心计算的留白让密集信息不显拥挤

**Color Philosophy**: 近乎纯白的暖色底（#FAFAF7）配深墨色文字（#1A1A2E），唯一的强调色是学术蓝（#2563EB）用于交互元素和重要数据。成功用墨绿（#065F46），警告用赭石（#92400E）。克制的用色传达"严谨学术"的气质。

**Layout Paradigm**: 报纸式多栏布局。主内容区采用不对称双栏（黄金比例），侧边信息栏窄而精。导航采用顶部水平条，像书的章节目录。

**Signature Elements**:
1. 手绘风格图表边框 — 数据可视化带有轻微的手绘质感
2. 脚注式提示 — 提示信息以学术脚注形式出现在页面底部
3. 书签式导航标记 — 当前页面在导航栏有一个折角书签效果

**Interaction Philosophy**: 极简的交互反馈。点击有墨水扩散效果，悬停仅改变下划线样式。一切交互都像在纸上做标记。

**Animation**: 几乎无动画，仅有极其克制的淡入（200ms）和轻微的位移（2px）。翻页效果模拟纸张翻转。

**Typography System**: 标题用 Playfair Display（衬线，学术感），正文用 Source Serif 4（优秀的屏幕衬线体），代码和数据用 Source Code Pro。

## Idea 3: "Glass Command Center" — 玻璃指挥中心

**Design Movement**: 受 Apple Vision Pro 和航天控制台启发的空间计算美学

**Core Principles**:
1. 玻璃态层叠 — 所有面板采用毛玻璃效果，通过透明度和模糊度区分层级
2. 空间深度感 — 利用阴影、模糊和微妙的视差创造三维空间感
3. 精密仪表盘 — 数据展示如同太空舱仪表，精确而优雅

**Color Philosophy**: 深色太空灰底色（#0F172A → #1E293B 渐变），玻璃面板用半透明白（rgba(255,255,255,0.05-0.1)）。主强调色为靛蓝（#6366F1），辅助用青色（#06B6D4）。成功用翡翠绿（#10B981），数据高亮用紫罗兰（#8B5CF6）。色彩系统传达"高科技控制中心"的沉浸感。

**Layout Paradigm**: 仪表盘式网格布局。左侧是可折叠的垂直导航塔（图标+文字），主区域是自适应网格，各功能模块如同控制台面板可以展开/折叠。顶部是状态栏显示关键指标。

**Signature Elements**:
1. 毛玻璃卡片 — 所有内容容器都有 backdrop-blur 和微妙边框发光
2. 环形进度仪表 — 能力值、XP等用精密的环形仪表展示
3. 光晕交互反馈 — 按钮和卡片悬停时有柔和的光晕扩散

**Interaction Philosophy**: 流畅的空间感交互。卡片悬停时轻微上浮并增加光晕，点击有涟漪扩散。侧边栏折叠/展开有弹性动画。模态框从远处"飞入"。

**Animation**: 使用 Framer Motion 的 spring 物理动画。入场动画有 stagger 延迟（每个元素间隔50ms）。图表数据用 ease-out 缓动绘制。背景有极其缓慢的渐变色流动（20s周期）。

**Typography System**: 标题用 Plus Jakarta Sans（现代几何无衬线，比 Inter 更有个性），正文用 Inter（最佳屏幕可读性），数据和代码用 Fira Code（带连字的等宽字体）。


---

## 选择方案

**选择 Idea 3: "Glass Command Center" — 玻璃指挥中心**

理由：
1. 深色主题 + 毛玻璃效果最适合学习平台的"沉浸式"体验
2. 仪表盘布局天然适配多功能学习工具
3. 精密仪表风格完美展示自适应算法的数据可视化
4. 用户偏好科技感设计和动画效果，此方案最契合
5. 可折叠侧边栏满足用户对单屏展示的需求
